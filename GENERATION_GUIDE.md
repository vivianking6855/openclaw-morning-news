## 二、晨报必备板块清单（校验机制）

### 必须包含的7大板块

- [ ] **1. 今日洞察** (hero-section)
  - 主标题：一句话概括今日重点
  - 摘要：3-4条核心要点

- [ ] **2. 务实落地建议** (recommendations)
  - 4条可操作建议

- [ ] **3. AI产业动态** (content-card)
  - 至少4条AI相关新闻
  - 标注P0/P1/P2优先级

- [ ] **4. 云原生/DevOps动态** (content-card)
  - K8s、平台工程等相关新闻

- [ ] **5. 具身智能/机器人** (content-card)
  - 融资、量产、技术突破

- [ ] **6. 核心摘要** (content-card full-width)
  - 今日核心观点总结

- [ ] **7. 大咖声音** (content-card full-width) ⭐ 高频遗漏
  - **必须包含上述11位大咖**
  - 每人一条近2年精华观点
  - 格式：头像/名字 + 职位 + 观点 + 时间/来源

## 三、校验脚本

生成晨报后，必须执行以下检查：

```bash
#!/bin/bash
# 晨报完整性校验

FILE=$1

echo "🔍 校验晨报完整性: $FILE"

# 检查必备板块
SECTIONS=("今日洞察" "务实落地建议" "AI产业动态" "核心摘要" "大咖声音")
for section in "${SECTIONS[@]}"; do
    if grep -q "$section" "$FILE"; then
        echo "✅ $section"
    else
        echo "❌ 缺失: $section"
        exit 1
    fi
done

# 检查11位大咖
EXPERTS=("巴菲特" "马斯克" "Altman" "黄仁勋" "梁文锋" "杨植麟" "凯文·凯利" "李彦宏" "雷军" "任正非" "贝索斯")
for expert in "${EXPERTS[@]}"; do
    if grep -q "$expert" "$FILE"; then
        echo "✅ $expert"
    else
        echo "⚠️ 未找到: $expert"
    fi
done

echo "✅ 校验通过"
```
